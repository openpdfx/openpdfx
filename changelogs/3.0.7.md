## OpenPDF 3.0.7 

### New Feature: PDF Page Rotation Utilities

**Added comprehensive page rotation support to `PdfBatchUtils`:**

* **`rotatePages(Path input, Path output, int rotation)`** - Rotate all pages in a PDF document by the specified angle (0, 90, 180, or 270 degrees). The rotation is applied cumulatively to any existing rotation.

* **`rotatePage(Path input, Path output, int pageNumber, int rotation)`** - Rotate a single specific page in a PDF document by the specified angle. Useful for fixing individual pages with incorrect orientation.

* **`batchRotatePages(List<RotateJob> jobs, Consumer<Path> onSuccess, Consumer<Throwable> onFailure)`** - Batch rotation of multiple PDF files using Java 21 virtual threads for efficient concurrent processing.

* **`RotateJob` record** - Configuration record for batch rotation operations containing input path, output path, and rotation angle.

**Key Features:**
- Supports standard rotation angles: 0, 90, 180, and 270 degrees
- Automatic normalization of negative and larger rotation values
- Cumulative rotation (adds to existing page rotation)
- Preserves all PDF metadata, annotations, and features
- Leverages Java 21 virtual threads for batch operations
- Comprehensive error handling with validation

**Example Usage:**

```java
// Rotate all pages in a PDF
Path rotated = PdfBatchUtils.rotatePages(
    Paths.get("input.pdf"), 
    Paths.get("output.pdf"), 
    90
);

// Rotate a single page
Path fixed = PdfBatchUtils.rotatePage(
    Paths.get("input.pdf"), 
    Paths.get("output.pdf"), 
    3,  // page number (1-based)
    180 // rotation angle
);

// Batch rotate multiple files
var jobs = List.of(
    new PdfBatchUtils.RotateJob(input1, output1, 90),
    new PdfBatchUtils.RotateJob(input2, output2, 180)
);

var result = PdfBatchUtils.batchRotatePages(jobs,
    path -> System.out.println("Rotated: " + path),
    error -> System.err.println("Failed: " + error)
);
```

This enhancement completes the core PDF manipulation utilities in `PdfBatchUtils`, alongside existing operations for merge, split, watermark, and encryption.
